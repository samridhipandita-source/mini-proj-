# mini-proj-
Problem statement - Write a 2d graphics editor using * and  _.  Write functions to draw a circle, rectangle, line, and triangle. You should be able to add objects to the picture, delete objects from the picture, and modify objects in the picture. Use a 2d array of characters to store the picture. Write a function to display the picture.
# 2D Graphics Editor in C

## Features
- Draw Point
- Draw Rectangle (Hollow/Filled)
- Draw Line (DDA Algorithm)
- Draw Triangle
- Draw Circle
- Delete Area
- Modify Area
- Clear Canvas
- Display Canvas

## Concepts Used
- 2D Character Arrays
- Functions
- Loops
- Conditional Statements
- Dynamic Canvas Size
- DDA Line Drawing Algorithm
- Circle Approximation
